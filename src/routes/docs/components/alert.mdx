## API

| Props | Default | Type | Description |
| --- | --- | --- | --- |
| title | `null` | string \| JSX.Element | Alert title |
| icon | `null` | JSX.Element | Icon to display in the alert |
| status | `info` | 'info' \| 'success' \| 'error' \| 'warning' | Alert status |
| size | `md` | 'sm' \| 'md' \| 'lg' | Alert size |
